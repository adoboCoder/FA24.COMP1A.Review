HTML - Media: Images, Audio, and Video

HTML provides elements for embedding multimedia content such as images, audio, and video into a webpage. This tutorial will cover how to use these elements effectively, providing examples and best practices for embedding media content in your HTML documents.

Embedding Images

Images are added to web pages using the <img> tag. This tag is a self-closing element that requires attributes to function correctly.

Example

Explanation
  • src: Specifies the path or URL of the image.
  • alt: Provides alternative text for the image, which is important for accessibility and is displayed if the image fails to load.
  • width and height: Define the image dimensions (optional).

Linking an Image

You can make an image clickable by wrapping it in an anchor (<a>) tag.

Example

When the image is clicked, the user will be directed to the URL specified in the <a> tag.

Responsive Images

To make images responsive (i.e., they resize according to the screen size), use CSS to control the image width relative to the screen size.

Example

The width: 100% ensures the image scales to the width of its container, and height: auto maintains the image's aspect ratio.

Embedding Audio

HTML5 introduced the <audio> element, which allows you to embed audio files directly into a webpage. You can use various file formats such as MP3, WAV, and Ogg.
